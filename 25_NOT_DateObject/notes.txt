#1] Date is an predefined object in javascript, just like math object which is also predefined object.


#2] When we want to make date object we use 'new' keyword along with it and make it with variable,
    Like,
    var d = new Date(); // we used 'new' keyword along with Date() object.


#3] Whenever we use date object, it basically shows us the our current day, date, time, time-zone by fetching it from our local computer,
    Like,
    var d = new Date();
    document.write(d);
    The above code will give output as,
    Sun Aug 20 2023 23:24:20 GMT+0530 (India Standard Time)
